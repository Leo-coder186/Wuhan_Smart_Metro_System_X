# 智慧地铁（Smart Metro）项目环境搭建指南

## 1. 前置环境准备

### 1.1 安装 Node.js

本项目基于 Vite 构建，推荐使用 **Node.js 18+** 或 **20+ LTS**。

- 下载地址：https://nodejs.org/
- 验证安装：

```bash
node -v
npm -v
```

### 1.2 安装 pnpm

项目使用 **pnpm** 作为包管理器（`.npmrc` 中配置了 `package-lock=false`）。

```bash
npm install -g pnpm
```

验证安装：

```bash
pnpm -v
```

## 3. 配置文件创建

### 3.1 .npmrc

在项目根目录创建 `.npmrc`，锁定使用 pnpm：

```ini
package-lock=false
```

### 3.2 vite.config.js

```js
import { defineConfig, loadEnv } from "vite";
import { fileURLToPath } from "url";
import vue from "@vitejs/plugin-vue";
import { viteExternalsPlugin } from "vite-plugin-externals";
import { insertHtml, h } from "vite-plugin-insert-html";
import { viteStaticCopy } from "vite-plugin-static-copy";
import compress from "vite-plugin-compression";

export default defineConfig(({ mode }) => {
  const envDir = "env";
  const isProd = mode === "production";
  const env = loadEnv(mode, envDir);
  const cesiumBaseUrl = env["VITE_CESIUM_BASE_URL"];
  const base = "/";

  const plugins = [
    vue(),
    // 生产环境将 Cesium 外部化（CDN 加载）
    viteExternalsPlugin({ cesium: "Cesium" }, { disableInServe: true }),
    // 注入 Cesium.js 脚本和样式
    insertHtml({
      head: [
        h("script", {
          src: isProd
            ? `${cesiumBaseUrl}Cesium.js`
            : `${base}${cesiumBaseUrl}Cesium.js`,
        }),
        h("link", {
          rel: "stylesheet",
          href: isProd
            ? `${cesiumBaseUrl}Widgets/widgets.css`
            : `${base}${cesiumBaseUrl}Widgets/widgets.css`,
        }),
      ],
    }),
  ];

  // 开发模式：从 node_modules 拷贝 Cesium 静态资源
  if (!isProd) {
    const cesiumRoot = "node_modules/cesium/Build/CesiumUnminified/";
    const dest = "libs/cesium/";
    const dirs = ["Assets", "ThirdParty", "Workers", "Widgets"];
    plugins.push(
      viteStaticCopy({
        targets: [
          { src: `${cesiumRoot}Cesium.js`, dest },
          ...dirs.map((d) => ({
            src: `${cesiumRoot}${d}/*`,
            dest: `${dest}${d}`,
          })),
        ],
      })
    );
  }

  plugins.push(compress({ threshold: 10 * 1024 }));

  return {
    base,
    envDir,
    mode,
    plugins,
    resolve: {
      alias: {
        "@": fileURLToPath(new URL("./src", import.meta.url)),
      },
    },
    css: {
      preprocessorOptions: {
        scss: {
          additionalData: '@import "@/assets/style/mixin.scss";',
        },
      },
    },
  };
});
```

### 3.3 环境变量

创建 `env/` 目录，存放环境变量文件：

**env/.env.development**

```env
VITE_BASE_URL='http://127.0.0.1:8090/api/v1'
VITE_CESIUM_BASE_URL='node_modules/cesium/Build/CesiumUnminified'
```

**env/.env.production**

```env
VITE_CESIUM_BASE_URL='https://your-cdn-path/libs/cesium/'
```

根目录 `.env` 作为开发默认值：

```env
VITE_BASE_URL='http://127.0.0.1:8090/api/v1'
VITE_CESIUM_BASE_URL='node_modules/cesium/Build/CesiumUnminified'
```

## 5. 核心代码实现

### 5.1 入口文件（src/main.js）

```js
import { createApp } from 'vue'
import Antd from 'ant-design-vue'
import 'ant-design-vue/dist/reset.css'
import App from './App.vue'
import { createPinia } from 'pinia'
import router from './router'
import VScaleScreen from 'v-scale-screen'
import './style.scss'

const app = createApp(App)
app.use(Antd).use(createPinia()).use(router).use(VScaleScreen).mount('#app')
```

### 5.2 根组件（src/App.vue）

```vue
<template>
  <v-scale-screen width="1920" height="1080" delay="200">
    <keep-alive>
      <CesiumView />
    </keep-alive>
  </v-scale-screen>
</template>

<script setup>
import CesiumView from './views/CesiumView.vue'
</script>
```

### 5.3 地图底图（src/cesiumTools/mapPlugin.js）

核心实现内容：
- **腾讯地图瓦片**：自定义 `TencentImageryProvider`（继承 `UrlTemplateImageryProvider`），支持卫星图和道路图
- **Google 地图瓦片**：自定义 `GoogleImageryProvider`
- **火星坐标系投影**：`GCJMercatorTilingScheme` 实现 GCJ-02 纠偏投影
- **坐标转换工具**：`CoordTransform` 类实现 WGS-84 ↔ GCJ-02 ↔ BD-09 互转

### 5.4 场景管理（src/cesiumTools/sceneManager.js）

核心接口：

| 函数 | 功能 |
|------|------|
| `initViewer(container)` | 初始化 Cesium Viewer，配置 WebGL，添加腾讯底图 |
| `setScene(viewer)` | 配置泛光、亮度、阴影、后处理等场景效果 |
| `loadTilesets(viewer, urls, cb)` | 批量加载 3DTiles 模型 |
| `handleDefaultModelEffect(tile)` | 使用 CustomShader 实现建筑高度渐变着色 |
| `flyToDefaultView(viewer)` | 飞至武汉默认视角 |

### 5.5 主视图（src/views/CesiumView.vue）

组件挂载时按顺序执行：
1. 获取 Cesium Ion Token
2. 初始化 Viewer
3. 获取天气数据（API 连通性测试）
4. 设置场景效果
5. 飞至默认视角（武汉）
6. 加载 3DTiles 建筑模型（`http://localhost:8888/wuhan/tileset.json`）
7. 给模型应用着色效果
8. 将 viewer 和 tileset 存入 Pinia store

### 5.6 路由（src/router/index.js）

```js
import { createRouter, createWebHashHistory } from "vue-router";

const routes = [];

const router = createRouter({
  history: createWebHashHistory(),
  routes,
});

export default router;
```

使用 Hash 模式，初始路由表为空（路由按需添加）。

### 5.7 状态管理（src/store/index.js）

定义两个 Store：
- **useLineData**：管理线路数据、Cesium Viewer 实例和 3DTiles 实例
- **useMeasureData**：管理测量工具数据

## 6. 运行项目

### 6.1 启动开发服务器

```bash
pnpm dev
```

项目默认运行在 `http://localhost:5173`。

### 6.2 生产构建

```bash
pnpm build
```

构建产物输出到 `dist/` 目录，包含：
- Vue 应用 bundle
- gzip 压缩后的静态资源（>10KB）
- 生产环境通过 CDN 加载 Cesium

### 6.3 预览构建结果

```bash
pnpm preview
```

```
├──────────────┴───────────────────────────────┤

├──────────────────────────────────────────────┤
│        API 层 (Axios + 线路接口)              │
└──────────────────────────────────────────────┘
```

### 7.1 Cesium 加载策略（开发 vs 生产）

| 阶段 | Cesium 加载方式 | 体积 |
|------|----------------|------|
| 开发 | `vite-plugin-static-copy` 从 node_modules 拷贝到 `libs/cesium/` | ~100MB（未压缩） |
| 生产 | `vite-plugin-externals` 外部化，通过 CDN 加载 | 不计入 bundle |

### 7.2 关键技术选型

- **Vite 5** 构建工具
- **CesiumJS 1.97** 3D 地图引擎
- **腾讯地图** 在线底图
- **3DTiles** 建筑白模（武汉地铁周边）
- **Sass/SCSS** 样式预处理
- **v-scale-screen** 大屏自适应方案（1920×1080 设计稿）
